# Приложение "конвертер валют"
![Логотип Конвертера валют](https://github.com/Waslerset/project_converter_valyt/blob/master/icon.jpg)
## Правила использования приложения
>1.Введите сумму для конвертации.
>
>2.Выберите валюту отправления и валюту получения.
>
>3.Дождитесь автоматического расчёта.
>
>**_Сайт с которого брали API иногда сбоит и курса некоторых валют нету, в таком случае повторить попытку конвертации данной валюты позже._**
